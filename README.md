📊 Customer Behavior Dashboard
🚀 Overview

The Customer Behavior Dashboard is an interactive analytics project designed to understand customer purchasing behavior, preferences, and trends.

This project combines SQL for data extraction, Python for preprocessing, and Power BI/Tableau for visualization to deliver meaningful business insights.

🎯 Key Features
📌 Total Customers Analysis (2.65K+)
💰 Average Purchase Amount ($59.54)
⭐ Customer Satisfaction Tracking (3.75 Rating)
🛒 Purchase History Insights (68K+ purchases)
📦 Category-wise Revenue & Sales Analysis
👥 Age Group Segmentation
🔍 Dynamic Filters:
Subscription Status
Gender
Product Category
Shipping Type
